#### 使用方式：

```bash
 sed -i '$a src-git mzwrt_package https://github.com/mzwrt/mzwrt_package' feeds.conf.default
```
<br>

#### 补丁应用
点击此链接[patch](https://github.com/mzwrt/mzwrt_package/wiki/patch)有详细解释，补丁是使用lean的lede进行编写的，默认支持所有openwrt主分支和其他分支
